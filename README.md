## Requirements

- Node (v18)

### Clone git project

```
git clone .....
```

### Configure environment variables

```
VITE_API_URL=http://localhost:8080
```

### Install dependencies

```
npm install
```

### Install json-server

```
npm install -g json-server
```

### run json server

```
json-server --watch db.json --port 8080
```

### run project

```
npm run dev
```
